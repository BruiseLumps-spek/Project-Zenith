# 🌌 Project Zenith: The Celestial Eye

> **Where Earth Meets the Cosmos.**

Project Zenith is an interactive real-time sky exploration platform that enables users to discover celestial objects visible from any location on Earth using live astronomical data. The platform combines immersive visualization, real-time satellite tracking, and intelligent exploration features to make astronomy engaging, educational, and accessible.

---

## 🚀 Features

* 🌍 Interactive Sky Explorer
* 🛰️ Live ISS Tracking
* 🌠 Real-Time Celestial Object Visualization
* ✨ Orbit Path Visualization
* 🤖 AI Astronomer (Interactive Celestial Guide)
* ⏳ Time Warp (Past & Future Sky Simulation)
* 🌌 Deep Space Journey
* ⭐ Constellation Explorer
* 📱 Responsive Cosmic-Themed Interface

* **CelesTrak API** – Satellite orbital information
* **OpenNotify API** – Live ISS position tracking

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Project-Zenith.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

Alternatively, host the project using GitHub Pages or Vercel for the best experience.
