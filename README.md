# 🌌 Project Zenith: The Celestial Eye

### *Where Earth Meets the Cosmos.*

Project Zenith: The Celestial Eye is an interactive real-time sky exploration platform designed to help users discover and visualize celestial objects visible from any location on Earth. By combining immersive 3D visualization with live astronomical data, the platform provides an engaging experience for students, educators, researchers, and space enthusiasts.

The application features a responsive cosmic-themed interface, an interactive 3D globe, live International Space Station (ISS) tracking, celestial object visualization, and AI-assisted astronomy guidance, making space exploration both educational and interactive.

# 📡 APIs Used

### 🌍 NASA Horizons API
Provides astronomical data including planetary positions and celestial object information.

### 🛰️ WhereTheISS.at API
Provides real-time International Space Station location, speed, altitude, latitude, and longitude.

### 🛰️ CelesTrak
Provides satellite orbital datasets for space object visualization.

### 🗺️ OpenStreetMap
Supports map-based location services and geographic visualization.
